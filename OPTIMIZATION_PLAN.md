- **前端**: React, JavaScript

## 注意事项

1. 保持后端运行在 `8001` 端口
2. 保持前端运行在 `3003` 端口
3. 每次修改后测试转录功能
4. 大改动前先创建备份分支
5. 已完成修改需提交 git
