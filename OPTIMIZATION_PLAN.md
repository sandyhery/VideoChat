# VideoChat 优化计划

> 创建时间: 2026-04-26
> 状态: 进行中

## 优化方向

### 1. 导出多模态分析结果为 md 格式 ✅ (优先级高)
- [ ] 详细总结导出为 Markdown
- [ ] 多模态分析结果导出为 Markdown
- [ ] 思维导图导出为 Markdown/Mermaid 格式
- [ ] 批量导出功能

### 2. 转录质量提升 (优先级高)
- [ ] 说话人分离 (Diarization)
- [ ] 音频降噪处理
- [ ] 支持更多语言
- [ ] 转录进度显示优化

### 3. 新增分析功能 (优先级中)
- [ ] 情感分析
- [ ] 关键词提取
- [ ] 视频片段搜索
- [ ] 字幕自动同步
- [ ] 语音时间戳优化

### 4. 后端优化 (优先级中)
- [ ] 添加缓存机制
- [ ] 异步任务队列
- [ ] API 限流保护
- [ ] 日志优化
- [ ] 错误重试机制

### 5. UI/UX 改进 (优先级低)
- [ ] 响应式设计优化
- [ ] 深色模式
- [ ] 转录进度动画
- [ ] 结果预览优化

## 技术栈

- **前端**: React, JavaScript
- **后端**: FastAPI, Python
- **STT**: FunASR (paraformer-zh), Faster-Whisper
- **AI**: MiniMax API

## 注意事项

1. 保持后端运行在 `8001` 端口
2. 保持前端运行在 `3003` 端口
3. 每次修改后测试转录功能
4. 大改动前先创建备份分支
