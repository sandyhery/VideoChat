# VideoChat (Airmomo) 多模态视频分析系统

VideoChat 是一个强大的视频分析系统，支持视频转录、总结生成、思维导图创建以及多模态分析功能。本系统结合了音频转录、视频截图分析和OCR文字识别技术，为用户提供全面的视频内容理解能力。

## 功能特性

### 核心功能
- **视频/音频转录**：支持多种格式的视频和音频文件转录
- **简单总结**：生成视频内容的简短总结
- **详细总结**：生成详细的视频内容分析报告
- **思维导图**：创建视频内容的思维导图
- **对话交互**：基于视频内容进行问答交互
- **多模态分析**：结合视频截图、OCR文字识别和音频转录进行综合分析

### 多模态分析特性
- **视频截图提取**：支持按时间间隔或场景变化提取视频截图
- **OCR文字识别**：识别截图中的文字内容
- **字幕检测**：专门检测视频中的字幕内容
- **多模态融合**：整合视频、音频和文字信息进行综合分析
- **智能分析**：使用AI模型生成综合分析报告和思维导图

## 系统架构

### 后端
- **FastAPI**：提供API端点
- **faster-whisper**：进行语音识别
- **OpenCV**：处理视频和图像
- **pytesseract**：进行OCR文字识别
- **ffmpeg**：处理视频文件
- **AI服务**：使用大模型进行分析和生成

### 前端
- **React**：前端框架
- **Ant Design**：UI组件库
- **jsMind**：思维导图可视化
- **ReactMarkdown**：Markdown渲染

## 安装和部署

### 环境要求
- Python 3.8+
- Node.js 14+
- 足够的存储空间（用于模型和缓存）
- 推荐使用GPU以提高转录速度

### 安装步骤

#### 1. 克隆仓库
```bash
git clone https://github.com/Airmomo/VideoChat.git
cd VideoChat
```

#### 2. 安装后端依赖
```bash
# 创建虚拟环境
python3 -m venv venv
# 激活虚拟环境
source venv/bin/activate  # macOS/Linux
# 或
venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt
```

#### 3. 安装前端依赖
```bash
cd frontend
npm install
```

#### 4. 配置AI API
复制配置文件模板并填写API密钥：
```bash
cp backend/config.template.py backend/config.py
```

编辑 `backend/config.py` 文件，填写您的AI API密钥：
```python
AI_CONFIG = {
    "api_key": "your-api-key",
    "base_url": "https://api.minimax.chat/v1/text/chatcompletions",
    "model": "MiniMax-M2.7",
    "max_tokens": 2000
}
```

#### 5. 启动服务

##### 启动后端服务
```bash
# 在VideoChat目录下
python app.py
```

##### 启动前端服务
```bash
# 在frontend目录下
npm start
```

## 使用指南

### 基本使用流程

1. **上传视频/音频文件**
   - 点击"上传本地文件"按钮
   - 选择要分析的视频或音频文件

2. **开始转录**
   - 在文件列表中选择要转录的文件
   - 点击"开始转录"按钮
   - 等待转录完成

3. **生成分析**
   - 选择已完成转录的文件
   - 切换到不同的标签页：
     - **简单总结**：生成简短的内容总结
     - **详细总结**：生成详细的分析报告
     - **思维导图**：创建内容的思维导图
     - **对话交互**：基于内容进行问答
     - **多模态分析**：进行综合分析

4. **导出结果**
   - 转录结果：支持导出为VTT、SRT、TXT格式
   - 总结：支持导出为Markdown格式
   - 思维导图：支持导出为JSON格式

### 多模态分析使用

1. **准备工作**
   - 确保视频已完成转录
   - 切换到"多模态分析"标签页

2. **开始分析**
   - 点击"开始多模态分析"按钮
   - 等待分析完成（分析过程可能需要几分钟，取决于视频长度）

3. **查看分析结果**
   - 分析完成后，系统会显示：
     - 视频基本信息
     - 截图数量
     - 综合分析总结
     - 分析思维导图

## 配置选项

### 后端配置

在 `backend/config.py` 文件中，您可以调整以下配置：

- **STT_CONFIG**：语音识别配置
  - `whisper_model`：Whisper模型（tiny, base, small, medium, large）
  - `beam_size`：波束搜索大小
  - `language`：语言设置
  - `vad_filter`：是否使用VAD过滤

- **AI_CONFIG**：AI服务配置
  - `api_key`：API密钥
  - `base_url`：API基础URL
  - `model`：使用的模型
  - `max_tokens`：最大token数

### 前端配置

在 `frontend/package.json` 文件中，您可以调整：

- `proxy`：后端API地址

## 性能优化

1. **模型选择**
   - 对于快速转录，使用 `tiny` 或 `base` 模型
   - 对于更高准确性，使用 `medium` 或 `large` 模型

2. **硬件加速**
   - 如果有GPU，系统会自动使用GPU加速
   - 在Mac上，系统会使用Metal Performance Shaders (MPS) 加速

3. **缓存机制**
   - 系统会缓存分析结果，提高重复分析的速度
   - 缓存大小默认为10个结果

## 故障排除

### 常见问题

1. **转录失败**
   - 检查视频文件格式是否支持
   - 确保模型下载成功
   - 检查系统资源是否充足

2. **多模态分析失败**
   - 确保视频文件存在
   - 检查转录结果是否完整
   - 查看后端日志了解具体错误

3. **OCR识别不准确**
   - 确保视频质量良好
   - 调整截图间隔，获取更清晰的画面

4. **系统性能问题**
   - 对于长视频，分析时间会相应增加
   - 考虑使用更小的模型以提高速度
